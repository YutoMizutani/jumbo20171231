# jumbo

2017年12月31日の年末ジャンボ当選確認プログラムです。Python3で動作します。
組と番号を入力すると当選の確認ができます。

「年末ジャンボ」「年末ジャンボミニ」「年末ジャンボプチ」の当選確認ができます。

# 起動方法

ダウンロード後，
- 年末ジャンボ -> jumbo.py
- 年末ジャンボミニ -> mini.py
- 年末ジャンボプチ -> puchi.py

を起動してください。
